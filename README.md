This addon for Sable Beyond adds dynamic mass for copycats from Create: Copycats+ mod. Example: copycat slab mass is 1 kpg, when iron block is applied mass will change to iron block's mass(4 kpg). It should work with all copycats and all blocks, any bugs can be reported at github page of the mod
